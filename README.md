Prepare (optional)
------------------

Set the base date by creating a file called `BASE`, with the format '%Y %m %d'.
The following is an example:
```
1990 01 01
```

Use the script
--------------

```bash
./duration
```

or

```bash
./duration 1990-01-01
```

or

```bash
Rscript duration.R 1990-01-01
```
